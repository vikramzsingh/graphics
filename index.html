Basic Shapes

SVG path element:-

UpperCase --> Absolute

LowerCase --> Relative

Mx y:- Move or start from x=10 and y=10 (M10 10)

Lx y:- Draw a Line from M to L100 150

lx y:- Draw a Line from M to l100 150, l is (relative to M points) which means x=90+10=100, y=140+10=150

vy:- Draw vertical from current point, v-50 (means it draw in upward direction). Note: it works in opposite that we represent on paper system

Vy:- Draw vertical start V30 from top to bottom 30px

hx:- Draw horizontal 20 from current point

mx y:- move x-horizontal and y-vertical from current point
